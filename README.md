# Students' Performance Analysis

## Objective
Analyze student academic performance using marks and attendance data.

## Dataset
The dataset contains student marks, subjects, and attendance percentage.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Analysis Performed
- Data cleaning and preprocessing
- Subject-wise performance analysis
- Attendance vs marks comparison

## Key Insights
- Students with higher attendance generally scored better
- Certain subjects showed lower average performance
- A small group of students require academic support

## Conclusion
Attendance plays a significant role in student academic performance.
